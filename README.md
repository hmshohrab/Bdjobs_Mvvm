# Bdjobs_Mvvm
Very basic Kotlin implementation following MVVM architecture using the latest components available like Material Component, Retrofit, LiveData ViewModel and Epoxy.
### This app builds with: 

- Kotlin
- MVVM architecture
- Repository pattern
- Rxjava2 
- Room Persistence DB
- Retrofit 2
- Material Design
- Epoxy
